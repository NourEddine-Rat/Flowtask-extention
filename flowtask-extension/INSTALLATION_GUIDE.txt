FlowTask Chrome Extension - Installation & Publishing Guide
=========================================================

## LOCAL INSTALLATION (Developer Mode)

### Step 1: Prepare Files
1. Make sure all files are in the 'flowtask-extension' folder:
   - manifest.json
   - index.html
   - styles.css
   - app.js
   - icons/ folder with SVG files
   - README.md

### Step 2: Install Locally
1. Open Chrome browser
2. Go to: chrome://extensions/
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Select the 'flowtask-extension' folder
6. Extension will appear in your extensions list
7. Open a new tab - you'll see FlowTask!

### Step 3: Test Features
- Add/edit/delete tasks
- Create timeline items
- Add sticky notes
- Toggle dark/light mode
- Add/edit custom quotes
- View history of deleted items
- Customize colors

### Step 4: Hide Browser Footer (Optional)
Some browsers show a customization footer at the bottom of new tab pages.
To hide it:

**Brave Browser:**
1. Click "Customize Brave" at the bottom of new tab
2. Go to "Footer" section
3. Toggle OFF "Show footer on New Tab page"

**Chrome Browser:**
1. Click "Customize Chrome" at bottom right
2. The footer auto-hides after customization

**Edge Browser:**
1. Click gear icon or "Page settings"
2. Adjust layout preferences

## CHROME WEB STORE PUBLISHING

### Prerequisites
- Google Developer Account ($5 one-time fee)
- ZIP file of extension
- Store listing assets (screenshots, descriptions)

### Step 1: Create Developer Account
1. Go to: https://chrome.google.com/webstore/devconsole
2. Sign in with Google account
3. Pay $5 registration fee
4. Accept developer agreement

### Step 2: Prepare Extension Package
1. Create PNG icons (Chrome requires PNG, not SVG):
   - 16x16, 48x48, 128x128 pixels
   - Use online converter for SVG to PNG
2. Update manifest.json to use .png files:
   "icons": {
     "16": "icons/icon16.png",
     "48": "icons/icon48.png", 
     "128": "icons/icon128.png"
   }
3. ZIP the entire extension folder

### Step 3: Create Store Listing
1. In Developer Console, click "New Item"
2. Upload ZIP file
3. Fill out store listing:

### Step 4: Add Screenshots
Required screenshots (1280x800 or 640x400):
1. Main dashboard view (light mode)
2. Dark mode view
3. Task management in action
4. Timeline feature
5. Notes and quotes sections

### Step 5: Set Categories & Metadata
- Category: Productivity
- Language: English
- Regions: All regions (or specific ones)

### Step 6: Privacy & Permissions
1. Add privacy policy (required):
   "FlowTask stores all data locally on your device. No data is collected, transmitted, or shared with third parties."

2. Justify permissions:
   - "storage": To save your tasks, notes, and preferences locally
   - "chrome_url_overrides": To replace the new tab page

### Step 7: Submit for Review
1. Click "Submit for review"
2. Review process takes 1-7 days
3. You'll get email notification of approval/rejection

### Step 8: After Approval
- Extension goes live on Chrome Web Store
- Users can install via store link
- You can update by uploading new ZIP files

## UPDATING THE EXTENSION

### For Local Development:
1. Make changes to files
2. Go to chrome://extensions/
3. Click refresh icon on FlowTask extension
4. Changes will apply immediately

### For Published Extension:
1. Update version number in manifest.json
2. Create new ZIP file
3. Upload to Developer Console
4. Submit for review again

## TROUBLESHOOTING

### Common Issues:
1. "Manifest file is missing" - Check file is named exactly "manifest.json"
2. "Icons not loading" - Ensure PNG files exist in icons/ folder
3. "Extension not working" - Check browser console for JavaScript errors
4. "Store rejection" - Usually due to policy violations or missing privacy policy

### Getting Help:
- Chrome Extension Documentation: https://developer.chrome.com/docs/extensions/
- Stack Overflow: Tag questions with "chrome-extension"
- Chrome Web Store Developer Support
