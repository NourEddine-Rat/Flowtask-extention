SINGLE PURPOSE DESCRIPTION (for Chrome Web Store):
This extension replaces the new tab page with a productivity dashboard that allows users to manage tasks, plan their day with a timeline, take quick notes, and view inspirational quotes.

PERMISSION JUSTIFICATIONS:
--------------------------
"storage" permission:
Used to save user's tasks, notes, timeline entries, quotes, theme preferences, and color customizations locally on their device. No data is transmitted externally.

"chrome_url_overrides" (newtab):
Required to replace the default new tab page with the FlowTask productivity dashboard interface.
